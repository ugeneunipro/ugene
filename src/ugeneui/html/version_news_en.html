Changes in UGENE v. 40

Alignment Editor

  • Multi-row selection support.
  • The "Build Tree" dialog now has all PhyML algorithms listed.
  • A new option to move selected sequences from one document to another.
  • The grouping mode: sorting by the size of a group and showing group sizes in the name list.

Sanger Editor

  • A new visualization mode: show alternative mutations.
  • Vertical scrolling bugs have been fixed.

Sequence Viewer

  • Improved speed, memory usage and precision of sequence graphs.
  • Fixed stability problems of auto-annotation routines.
  • Sequences with sizes above 2Gb now can be opened by UGENE again.

Other improvements

  • Improved CUDA build and error detection support.
  • Unicode supported was added for Newick, Plain text, Diff, VCF, SNP file formats.
  • Random Sequence Generator memory footprint and generation speed improvements.

A complete list of changes can be found on the UGENE web site.
